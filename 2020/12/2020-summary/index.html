2020 年终总结

时间飞快，2020年最后一个月了也快要过完了，又到了总结自己这一年来的所有的经历和收获。

2020年初开始的新冠病毒，它就像网络病毒一样，从一点扩散到一个城市，再到一个省，然后是整过国家，最后是全世界都感染上了这个病毒。

无论是生活还是工作，每个人都被影响着。

生活

本来春节期间和小伙伴约好了要去一趟美国自驾，以及看望我们共同的朋友，机票早就订好了，但疫情的一天天的严重，机场也成为了最为危险的地方，最终机票被自动退票了，这趟令我们期盼的旅行也被迫终止。

另外一个

2020年全世界人都经历了太多，疫情、在家办公、戴口罩、

从中国新年腊月二十几开始，

工作之内

年底了，也到了写工作总结的时候了。应该很多人都不太愿意去写工作总结，我也是“被迫”的，但写总结可以倒逼我更好的回顾这一年来工作上的成绩，当写出来时发现时间的积累是惊人的，自己这一年做了很多事情，流水账就不写了，写几个重要的成果。

  1. 通过 Jenkins Shared Libraries和 Multi-branch Pipeline 实现了无人值守的自动化构建。

    • 不但节省了我的时间，也节省了开发的时间，满足他们的构建需求。
    • 让 Build Engineer 变成了 DevOps Engineer，去做更有价值的事。
    • 通过 Pull Request Build 的引入，在 Merge 前校验所有平台构建是否正确，从而保证主分支的稳定。

  2. 推动使用 Artifactory 来提供下载和存储构建。

    • 通过使用 Artifactory，进一步完善团队的 CI/CD 的自动化流水线。
    • 同时更好的为团队的其他自动化做基础，比如自动下载、安装、测试等。

  3. 其他

  • 这一年更多的是使用 Python 来做一些集成，包括做了很多 Jira, BitBucket 的集成。另外还了解了 Python 发布一个项目需要注意哪些，并为公司发布了 UOPY 这个项目。
  • 学习了一些新工具，比如 JaCoCo、Ansible playbook、ELK等

工作以外

依旧坚持在博客和公众号(DevOps攻城狮)分享一些工作中的技术总结。
